# TypeZilla

TypeZilla is a Next.js application designed to help users improve their typing speed and accuracy through engaging typing challenges and live performance feedback.

## Features

-   **Live Typing Metrics:** Get real-time updates on your Words Per Minute (WPM), accuracy, and elapsed time as you type.
-   **Randomized Texts:** Practice with a variety of sample texts to keep challenges fresh.
-   **Virtual Keyboard:** Visual feedback on a virtual keyboard shows which keys are being pressed and highlights errors.
-   **Interactive Text Display:** See your typed characters, current position, and errors highlighted directly on the text.
-   **Responsive Design:** Works smoothly on different screen sizes.
-   **Modern UI:** Built with ShadCN UI components and Tailwind CSS for a clean and professional look.

## Tech Stack

-   **Next.js:** React framework for server-side rendering and static site generation.
-   **React:** JavaScript library for building user interfaces.
-   **TypeScript:** Superset of JavaScript for type safety.
-   **ShadCN UI:** Re-usable UI components.
-   **Tailwind CSS:** Utility-first CSS framework for styling.
-   **Lucide Icons:** For clean and consistent iconography.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (version 18.x or later recommended)
-   npm or yarn

### Installation

1.  **Clone the repo:**
    ```sh
    git clone https://github.com/MalikRehan0606/TypeZilla.git
    ```
    (Replace `TypeZilla.git` with your actual repository name if different)
2.  **Navigate to the project directory:**
    ```sh
    cd TypeZilla
    ```
3.  **Install NPM packages:**
    ```sh
    npm install
    ```
    or if you use yarn:
    ```sh
    yarn install
    ```

### Running the Development Server

Execute the following command to start the development server:

```sh
npm run dev
```

or

```sh
yarn dev
```

Open [http://localhost:9002](http://localhost:9002) (or the port specified in your `package.json` `dev` script) with your browser to see the result.
