# About TypeZilla

TypeZilla is a modern web application designed to help users improve their typing speed and accuracy in an engaging and interactive way. Built with Next.js and TypeScript, it provides a feature-rich platform for typists of all skill levels.

## Key Features

*   **Varied Typing Challenges**: Users can choose from different modes, including word-based tests with simple, intermediate, and expert difficulties, or time-based challenges.
*   **Real-time Performance Metrics**: Get instant feedback on your Words Per Minute (WPM), accuracy, and consistency as you type.
*   **AI-Powered Analysis**: After each test, an AI coach provides personalized feedback on your performance, highlighting strengths and offering actionable tips for improvement.
*   **Global Leaderboards**: Compete with other users worldwide. Leaderboards can be filtered by test type and timeframes (daily, weekly, all-time).
*   **User Profiles & Progress Tracking**: Registered users have personal profiles that track their stats, showcase personal bests, and display performance history over time.
*   **Social Features**: Users can view each other's public profiles and send friend requests to build a community.
*   **Customizable Experience**: Settings allow users to change the app's language, typing sounds, and visual themes.
